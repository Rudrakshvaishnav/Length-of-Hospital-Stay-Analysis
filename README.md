# Length of Hospital Stay Analysis

## 📌 Overview

This project analyzes the factors affecting hospital length of stay using data analytics and machine learning techniques.

## 🎯 Objective

To identify key factors influencing hospital stay duration and build a predictive model.

## 📊 Dataset

* Patient demographics (Age, Gender)
* Medical conditions
* Admission details
* Treatment information

## ⚙️ Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building (Random Forest)
5. Evaluation

## 📈 Key Insights

* Medical condition significantly affects hospital stay
* Readmission leads to longer stays
* No single feature strongly predicts LOS
* Healthcare data is complex and multi-factor dependent

## 🤖 Model Performance

* R2 Score: ~ -0.03
* MAE: ~ 17 days

## 🛠️ Tools Used

* Python
* Pandas
* Seaborn
* Scikit-learn
