# Length-of-Hospital-Stay-Analysis

## Overview
This project analyzes hospital length of stay using data analytics and machine learning.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Machine Learning Model (Random Forest)
- Feature Importance Analysis

## Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn

## Results
The model achieved realistic performance, highlighting the complexity of healthcare data.
